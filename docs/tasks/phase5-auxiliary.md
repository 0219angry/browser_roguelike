# Phase 5: 補助システム

企画書 §17.2 Phase 5 に対応。ショップ・補給端末・経済バランスを整える。

## タスク一覧

### 5-1. メインショップ
- 専用フロアまたはフロア間報酬画面として出現（3フロアに1回程度）
- 販売候補: 遺物 / 装備 / 消耗品 / 回復 / 再抽選 / 呪い除去
- Scrap（通貨）による購入処理
- 役割: ランの歪みを補正する場。大量購入の場ではない
- 参照: §12.3

### 5-2. 補給端末（盤面オブジェクト）
- フロア盤面上に 20% の確率で出現（§19）
- 1回使い切り、利用時に危険度 +1
- 提供サービス例（小規模）:
  - 消耗品を1つ購入
  - 小回復
  - スキルチャージ回復
  - その場で候補を再抽選
- 階段付近への配置禁止
- 参照: §12.4

### 5-3. Scrap（通貨）管理
- 敵撃破・箱・イベントで Scrap を取得
- ショップ・補給端末・再抽選で消費
- Player エンティティの `scrap` フィールドと連動
- 参照: §12.1

### 5-4. 報酬・経済バランス調整
- 1フロアあたりの Scrap 収入期待値を設定
- ショップ各アイテムの価格テーブルを設定
- 再抽選コストの設定
- 参照: §12.2〜12.4

### 5-5. サブ目標
- フロアに 0〜1 個配置（任意達成）
- 達成で追加報酬（Scrap, 消耗品など） + 危険度 +1
- 例: 発電機2基破壊 / 制圧マスに2ターン立つ / エリートを倒す / 一定ターン以内に階段到達
- 「やらない自由」を保証（進行必須にしない）
- 参照: §3.4

## 完了条件
- ショップと補給端末が機能し、Scrap の収支が自然に成立している状態
