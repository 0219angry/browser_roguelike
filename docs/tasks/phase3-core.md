# Phase 3: 本作の核成立

企画書 §17.2 Phase 3 に対応。「まだ残るか、ここで降りるか」の判断が生まれる。

## タスク一覧

### 3-1. 危険度（Danger）システム
- 危険度カウンターの実装と UI 表示
- 上昇トリガーの実装:
  - プレイヤーターン終了 +1
  - 敵撃破 +1
  - サブ目標達成 +1（Phase 5 で接続）
- 危険度フェーズ判定（Calm / Alert / Crisis / Collapse / Death）
- 参照: §5.1〜5.4、§19

### 3-2. 危険度フェーズ効果
- **Alert (3〜5)**: 敵索敵距離増加、一部スポナー起動
- **Crisis (6〜8)**: 毎2ターンに増援1体、危険地形1〜2箇所追加
- **Collapse (9〜11)**: 一部敵が追加能力獲得、毎ターン危険地形拡大
- **Death (12+)**: エリート増援、危険地形拡大強化
- 参照: §5.3〜5.4

### 3-3. 増援システム
- 危険度に応じて盤面外縁にエネミーをスポーンさせる
- 増援敵の報酬倍率を設定（25%）
- 危険度9以上の増援は EXP 0
- `source: 'reinforcement'` フラグを Enemy に付与
- 参照: §5.7、§14.1（Enemy エンティティ）

### 3-4. 階段・フロア遷移
- 階段マス上でのフロア移動アクション（コスト 1 AP）
- フロア開始時から階段は可視・使用可能
- 遷移時の処理: 危険度リセット、フロア後回復の適用
- 参照: §2.1、§3.1、§4.2

### 3-5. フロア後回復
- フロア移動時に最大HPの10%回復
- 危険度3以下での離脱: さらに +10%
- 危険度8以上での離脱: 追加回復なし
- 参照: §6.5、§7.5

### 3-6. フロア報酬提示
- フロア遷移後に報酬選択画面を表示
- 基本は **3択提示**
- Phase 4 でレベルアップ・遺物・装備と接続
- 参照: §7.4

### 3-7. 報酬逓減
- 経験値効率カーブの実装:
  - 1〜3体目: 100%
  - 4〜6体目: 70%
  - 7体目以降: 40%
  - 増援: 25%
  - 危険度9以上の増援: 0%
- 参照: §5.7

## 完了条件
- 危険度が上がるにつれてフロアが圧力を増し、階段から降りることに明確なメリットがある状態
