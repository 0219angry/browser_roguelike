# Phase 4: ラン成立

企画書 §17.2 Phase 4 に対応。1ランを通してビルドが育ち、クリアまで遊べる状態にする。

## タスク一覧

### 4-1. 経験値・レベルアップ
- 敵撃破時の経験値付与（報酬逓減適用済み）
- レベルアップ必要 EXP 設定（Lv2:8 / Lv3:18 / Lv4:32 / Lv5:50 / Lv6:72）
- レベルアップ時に **3択選択画面** を表示
- 候補プール（最低以下を実装）:
  - 最大HP +4
  - 通常攻撃 +1
  - 防御時に隣接敵へ1ダメージ
  - 移動後の最初の攻撃にノックバック
  - 危険地形ダメージ無効
  - 危険度3以下で離脱した際の追加回復 +5%
- 参照: §6.3、§6.4、§19

### 4-2. 遺物システム（15個）
- `trigger + condition + effect` の構造で定義
- MVPで実装する15個の候補（§8.3 より）:
  - 行動変換系: 反響ブーツ / 迎撃の護石 / 観測レンズ
  - 条件報酬系: 残火炉 / 狩人の印
  - リスク交換系: 過充電心臓 / 割れた聖杯
  - 危険度活用系: 緊急駆動片 / 深部共鳴器
  - ルール変形系（レア）: 逆位の羅針盤 / 透過靴
  - 残り5個は調整フェーズで追加
- 参照: §8.1〜8.3

### 4-3. 装備システム（9個）
- 装備スロット: 武器 / 靴 / 護符 の3枠
- MVPで実装する9個（§8.5 より各スロット3個）:
  - 武器: 反動槌 / 突槍 / 双刃
  - 靴: 踏破靴 / 跳躍靴 / 残像靴
  - 護符: 返礼の護符 / 偏差の護符 / 節制の護符
- 参照: §8.5

### 4-4. スキル改造（8個）
- 初期スキル: Dash / Guard / Pulse / Throw
- スキルは `baseSkill + mods[]` 構造で定義
- MVPで実装する改造（§9.3 より各スキル2個、計8個）:
  - Dash: 敵すり抜け可能 / 階段方向への距離+1
  - Guard: 成功時に反撃 / 危険度6以上ならシールド追加
  - Pulse: 壁衝突時に追加ダメージ / 危険地形上の敵に距離+1
  - Throw: 着弾地点に危険地形生成 / 行動予告中の敵にダメージ増加
- 参照: §9.1〜9.4

### 4-5. 消耗品（6個）
- 消耗品スロット 2 枠
- MVPで実装する6個（§8.4 より）:
  - 煙幕筒 / 時止め片 / 磁気地雷 / 応急パッチ
  - 残り2個は調整フェーズで追加
- 参照: §8.4

### 4-6. ボス実装（中ボス1種 + ボス1種）
- 中ボス: 特殊ルール付きの強化敵（独自 intent パターン）
- ボス: 最終フロア固定、クリア条件に接続
- MVPラン構成: 6フロア + ボス（§15.1）
- 参照: §2.2、§3.3（Boss フロア）、§15.1

### 4-7. ラン構成・フロア生成
- MVPラン: 6フロア + ボスフロア
- フロア種別4種の生成: standard / pressure / bounty / cache
- 生成制約の実装（§11.1〜11.3）:
  - 入口と階段を離す
  - 障害物率 18〜28%
  - 初期敵 5〜8体
  - 禁止事項（階段完全封鎖など）の回避
- 参照: §11、§15.1

## 完了条件
- 7フロア（6 + ボス）を通してビルドが変化し、ボスを倒してクリアできる状態
